## index.html 功能總覽（台灣氣象助理）

### 1. 主要目標
- 提供台灣縣市/行政區的即時、觀測、預報之氣象資訊。
- 顯示關鍵天氣要素（溫度、降雨機率、AQI、UV、能見度、風速等）並提供 AI 建議摘要。
- 支援地理定位，依據使用者座標自動切換到最近的測站所在縣市/行政區。

### 2. 資料來源與模式
- **真實資料模式**：使用中央氣象署 CWA API  
  - 觀測：`O-A0001-001`（TEMP/HUMD/WIND_SPEED/VISB/WEATHER）  
  - 預報：`F-C0032-001`（36 小時預報：MinT/MaxT/PoP/Wx）
- **模擬資料模式**：未設定 API Key 時自動回退，生成隨機但合理的氣象數據與時間。

### 3. 更新時間顯示邏輯
- Footer 顯示「更新時間」，資料完全源自 CWA：  
  1) 觀測站 `ObsTime.DateTime`（同縣市優先，其次任一站）  
  2) 預報發布時間 `datasetPublishTime`  
  3) 預報第一時段的 `startTime/dataTime/endTime`
- 模擬模式則使用本機時間，但會標記「使用模擬資料」。

### 4. 介面與互動
- **城市/行政區選擇**：下拉選單切換縣市與行政區。
- **分頁**：`現在` / `觀測` / `預報` 三個 Tab 切換不同視圖。
- **定位按鈕**：使用瀏覽器 Geolocation，反查最近測站並切換 UI 選擇。
- **資料卡片**：以卡片方式呈現即時要素，並用色彩/圖示強調狀態（如 AQI、UV）。

### 5. 天氣圖示與狀態
- 內嵌 SVG 圖示元件（晴、雲、雨等），並有 `WeatherIcon` 根據天氣型態顯示對應圖示。
- 依預報/觀測文字自動選定 `Sunny` / `Cloudy` / `Rain` 圖示。

### 6. AI 建議摘要
- 根據溫度、是否下雨、UV、風速、AQI、能見度等要素，產生最多兩則提示；若無警示則提供簡要建議（如補水、保暖）。

### 7. 其他細節
- 日出日落：以季節性估計（北/南部簡化模型）提供近似時間。
- 介面樣式：Tailwind + 自訂背景漸層，行動優先（最大寬 480px），卡片/圓角/陰影提升可讀性。
- 單位提示：預報頁底部顯示單位 `°C`，避免誤讀。

### 8. 主要依賴
- React 18（UMD）、Babel（線上轉譯 JSX）、Tailwind CDN。
- 瀏覽器 Geolocation API。
